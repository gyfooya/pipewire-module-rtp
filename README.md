# pipewire-module-rtp
UDP network audio stream

# What this gives you

- ✔ UDP audio streaming
- ✔ LAN real-time audio
- ✔ Works with any app (system-wide)
- ✔ No DAW dependency
- ✔ Can be routed like virtual cables
